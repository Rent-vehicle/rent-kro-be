# Bike & Car Rental Marketplace (AdonisJS Backend)

This project is the backend system for a vehicle rental marketplace.  
It allows **users** to rent cars/bikes, **owners** to list their vehicles, and **admins** to manage the platform.

---

## 🚀 Tech Stack
- [AdonisJS 5](https://adonisjs.com/) - Node.js MVC framework
- PostgreSQL - Database
- AWS - Cloud Hosting
- Lucid ORM - Database modeling
- TypeScript

---

## 📌 Features (MVP)
- User authentication (signup/login) (to be done)
- Vehicle listing & search (to be done)
- Booking system (to be done)
